# 🚀 Windows Optimization Toolkit - Execução Remota

Otimize seu Windows diretamente via PowerShell, sem precisar baixar arquivos!

## ⚡ Execução Rápida (Copie e Cole)

Abra o **PowerShell como Administrador** e execute:

```powershell
iwr -useb https://raw.githubusercontent.com/SEU_USUARIO/windows-optimization-toolkit/main/optimize.ps1 | iex
```

## ✨ O que o script faz:

- ✅ Ativa o Plano de Desempenho Máximo
- ✅ Limpa arquivos temporários
- ✅ Configura DNS Cloudflare (1.1.1.1) para melhor velocidade
- ✅ Cria ponto de restauração automático
- ✅ Otimiza interfaces de rede

## ⚠️ Requisitos:

- Windows 10 ou Windows 11
- PowerShell 5.0+
- Privilégios de Administrador

## 🔒 Segurança:

O script é 100% transparente e pode ser revisado antes da execução. Clique no arquivo `optimize.ps1` acima para ver o código-fonte completo.
