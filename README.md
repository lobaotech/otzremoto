# 🚀 Windows Optimization Toolkit - Menu Interativo Remoto

Otimize seu Windows com um menu interativo, baixando e executando scripts diretamente do GitHub!

## ⚡ Execução Rápida (Copie e Cole)

Abra o **PowerShell como Administrador** e execute:

```powershell
iwr -useb https://raw.githubusercontent.com/lobaotech/otzremoto/main/menu.ps1 | iex
```

## ✨ Como funciona:

1.  O comando acima baixa e executa o script `menu.ps1` diretamente do GitHub.
2.  Um menu interativo será exibido no PowerShell, permitindo que você selecione os módulos de otimização desejados.
3.  Ao confirmar, o script baixará e executará apenas os scripts selecionados, em tempo real.

## 📂 Módulos de Otimização Disponíveis (Pack Premium):

-   **Preparação e Backup**: Crie pontos de restauração e backup do registro antes de otimizar.
-   **Otimização de CPU e Energia**: Ajustes para máximo desempenho do processador e planos de energia.
-   **Otimização de GPU e Vídeo**: Otimizações para placas de vídeo NVIDIA, AMD e Intel.
-   **Rede e DNS**: Redução de latência de rede e configuração de DNS otimizado.
-   **Input Lag e Periféricos**: Minimização de atrasos em mouse, teclado e outros periféricos.
-   **Memória RAM**: Ajustes para melhor gerenciamento e uso da memória.
-   **Debloater e Privacidade**: Remoção de aplicativos indesejados e telemetria do Windows.
-   **Limpeza e Manutenção**: Limpeza profunda de arquivos temporários e otimização do sistema.
-   **Inicialização e Serviços**: Otimização do tempo de boot e gerenciamento de serviços.
-   **Otimizações por Jogo**: Scripts específicos para jogos populares como CS2, Valorant, Fortnite, etc.
-   **Reparo e Diagnóstico**: Ferramentas para verificar e reparar a integridade do sistema.

## ⚠️ Requisitos:

-   Windows 10 ou Windows 11
-   PowerShell 5.0+
-   Privilégios de Administrador
-   Conexão com a internet

## 🔒 Segurança:

Todos os scripts são transparentes e podem ser revisados no repositório GitHub. O script `menu.ps1` cria um ponto de restauração antes de aplicar as otimizações, garantindo a segurança do seu sistema.
